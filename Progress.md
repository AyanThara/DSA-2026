## 📅 Day 15 – Sorting Algorithms
###  Topics Covered

- Best Case, Worst Case, Average Case
- In-place sorting
- Stable vs Unstable sorting

## Selection Sort
### Concept
- Select minimum element from unsorted part
- Swap it with current index
- After each round, one element gets fixed
### Complexity
- Best Case: O(n²)
- Average Case: O(n²)
- Worst Case: O(n²)
- Space: O(1)
- Stable:  No
- In-place:  Yes



## Bubble Sort
### Concept
- Compare adjacent elements
- Swap if left > right
- After each round, largest element moves to end
### Complexity
- Best Case: O(n) (Optimized)
- Average Case: O(n²)
- Worst Case: O(n²)
- Space: O(1)
- Stable:  Yes
- In-place:  Yes


## Insertion Sort
###  Concept
- Pick element
- Shift bigger elements right
- Insert element at correct position
### Complexity
- Best Case: O(n)
- Average Case: O(n²)
- Worst Case: O(n²)
- Space: O(1)
- Stable:  Yes
- In-place:  Yes
